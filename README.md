# Super Store Sales Dashboard
## Project Overview

This project presents an interactive Super Store Sales Dashboard created using Power BI to analyze sales performance and business trends.

The dashboard helps businesses understand sales distribution, profit trends, product performance, and regional sales insights. It converts raw sales data into meaningful visualizations that support data-driven decision making.

## Objectives

The main objectives of this project are:

• Analyze overall sales performance

• Identify top performing products and categories

• Understand regional sales distribution

• Track sales and profit trends over time

• Provide interactive data visualization for business insights

## Dataset Description

The dataset used in this project contains information related to retail store sales such as:

• Order ID

• Order Date

• Customer Name

• Region

• State

• Product Category

• Sub Category

• Sales Amount

• Quantity Sold

• Profit

This dataset helps analyze business performance and identify sales patterns.

## Tools and Technologies Used

The following tools were used in this project:

• Power BI

• Data Modeling

• Data Visualization

• DAX (Data Analysis Expressions)

## Dashboard Features

The Super Store Sales dashboard includes the following visualizations:

• Total Sales Overview

• Total Profit Analysis

• Sales by Category

• Sales by Sub Category

• Region Wise Sales Distribution

• Monthly Sales Trends

• Top Performing Products

These visualizations allow users to explore business performance interactively.

## Key Insights

Some insights obtained from the dashboard include:

• Identification of the most profitable product categories

• Regions generating the highest revenue

• Monthly sales trends and seasonal patterns

• Products contributing the most to overall profit
